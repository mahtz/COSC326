# COSC326
All programs from COSC326 - Effective Programming Semester 2 2021, a paper in which we were tasked with 12 etudes (assignments) to complete.

These were:

1 - Mu Torere

2 - Fixing the Games

3 - Finding Anagrams

4 - Pentominoes

5 - Prime Days

6 - Preferential Voting

7 - Up or Down

8 - Woof

9 - Epidemic

10 - Transport Layer Security (TLS) Incident

11 - Contact Tracing

12 - Bug Squashing
